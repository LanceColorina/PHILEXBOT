to run the app locally:

create 2 terminals

1 - for client
2 - for server

For Client:

cd client/phillexbot
npm run dev 

For Server:

at the root folder -> .\venv\Scripts\Activate.ps1

python -m server.server


Note when installing libraries use:

.\venv\Scripts\Activate.ps1
python -m pip install <library>
